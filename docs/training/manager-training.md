# Manager Training Guide

## Overview
This training covers manager-specific features including approvals, team management, and reporting.

## Session 1: Manager Dashboard (30 minutes)

### Objectives
- Understand manager dashboard
- View team overview
- Access pending approvals

### Activities
1. Explore dashboard
2. View team members
3. Check pending approvals

## Session 2: Leave Approvals (60 minutes)

### Objectives
- Approve/reject leave requests
- Understand multi-level approvals
- Monitor SLA compliance

### Activities
1. Review pending leave requests
2. Approve/reject requests
3. Add comments
4. Track approval chain
5. Handle escalations

## Session 3: Team Management (45 minutes)

### Objectives
- View team profiles
- Monitor team attendance
- Manage team leaves

### Activities
1. View team member profiles
2. Check team attendance
3. View team leave calendar
4. Identify conflicts

## Session 4: Reports & Analytics (45 minutes)

### Objectives
- Generate team reports
- Analyze attendance trends
- Review leave patterns

### Activities
1. Generate leave utilization report
2. Create attendance summary
3. View analytics dashboard
4. Export reports

## Session 5: Delegation (30 minutes)

### Objectives
- Set up delegation
- Manage delegations
- Approve on behalf

### Activities
1. Create delegation
2. View active delegations
3. Approve delegated requests

## Assessment

### Quiz Questions
1. How do you approve a leave request?
2. What is SLA in approvals?
3. How do you set up delegation?

### Practical Exercise
- Approve test leave requests
- Generate team report
- Set up delegation
