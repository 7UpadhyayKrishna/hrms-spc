# Employee Onboarding Training

## Welcome to HRMS

This training guide will help you get started with the HRMS system.

## Session 1: Getting Started (30 minutes)

### Objectives
- Understand system navigation
- Complete profile setup
- View leave balance

### Activities
1. Login to system
2. Explore dashboard
3. Complete profile
4. View leave balance

## Session 2: Leave Management (45 minutes)

### Objectives
- Apply for leave
- Track leave requests
- View leave calendar

### Activities
1. Apply for leave
2. Track approval status
3. View leave calendar
4. Practice editing/withdrawing requests

## Session 3: Attendance & Work Schedule (30 minutes)

### Objectives
- View attendance records
- Understand work schedule
- Submit regularization requests

### Activities
1. View attendance records
2. Check work schedule
3. Submit regularization request

## Session 4: Profile Management (30 minutes)

### Objectives
- Update profile information
- Add family details
- Manage certifications

### Activities
1. Update personal information
2. Add family member
3. Add certification
4. Submit profile update request

## Assessment

### Quiz Questions
1. How do you apply for leave?
2. Where can you view your leave balance?
3. How do you update your profile?

### Practical Exercise
- Apply for a test leave request
- Update profile information
- View attendance records
