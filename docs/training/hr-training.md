# HR Training Guide

## Overview
This comprehensive training covers all HR functions including employee management, leave accrual, approval workflows, and system configuration.

## Session 1: Employee Management (60 minutes)

### Objectives
- Onboard new employees
- Manage employee profiles
- Process offboarding

### Activities
1. Create new employee
2. Set up employee profile
3. Configure work schedule
4. Process offboarding

## Session 2: Leave Accrual Configuration (90 minutes)

### Objectives
- Configure accrual policies
- Process monthly/yearly accrual
- Handle year-end processing

### Activities
1. Create accrual policy
2. Configure pro-rata and carry forward
3. Process monthly accrual
4. Process year-end carry forward

## Session 3: Approval Workflows (90 minutes)

### Objectives
- Create approval workflows
- Configure approval matrix
- Manage delegations

### Activities
1. Create multi-level workflow
2. Configure approval matrix
3. Set up delegation
4. Test workflow

## Session 4: Work Schedule Management (60 minutes)

### Objectives
- Create shift templates
- Manage rosters
- Handle roster changes

### Activities
1. Create shift template
2. Assign work schedules
3. Bulk upload rosters
4. Approve roster changes

## Session 5: Biometric Integration (60 minutes)

### Objectives
- Register biometric devices
- Sync employees
- Pull attendance data

### Activities
1. Add biometric device
2. Sync employees to device
3. Pull attendance data
4. Validate records

## Session 6: Reports & Analytics (60 minutes)

### Objectives
- Generate comprehensive reports
- Use analytics dashboard
- Configure scheduled reports

### Activities
1. Generate leave reports
2. Create attendance reports
3. View analytics
4. Set up scheduled reports

## Session 7: SAP Integration (90 minutes)

### Objectives
- Configure SAP connection
- Sync employee data
- Sync leave balances

### Activities
1. Configure SAP connection
2. Sync employee master data
3. Sync leave balances
4. Review sync logs

## Assessment

### Quiz Questions
1. How do you configure leave accrual?
2. How do you create an approval workflow?
3. How do you sync biometric data?

### Practical Exercise
- Configure complete leave management setup
- Create approval workflow
- Generate and schedule reports
