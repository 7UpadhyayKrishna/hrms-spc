## Getting Started

### Login
1. Navigate to the HRMS login page
2. Enter your email address and password
3. Click "Sign In"

### Dashboard Overview
Upon login, you'll see your employee dashboard with:
- Personal information summary
- Leave balance overview
- Upcoming holidays
- Pending approvals (if applicable)
- Recent activity

## Profile Management

### Viewing Your Profile
1. Click on "Profile" in the navigation menu
2. View your personal, employment, family, certifications, and education details

### Updating Profile Information
1. Navigate to your Profile
2. Click "Edit" button
3. Make necessary changes
4. Click "Save"
5. For certain fields, changes require approval (see Profile Update Requests)

### Profile Update Requests
For fields that require approval:
1. Make changes in the profile
2. Submit an update request
3. Track request status in "Update Requests" tab
4. Once approved, changes will be reflected in your profile

### Family Details
1. Go to Profile → Family Details tab
2. Click "Add Family Member"
3. Fill in relationship, name, date of birth, and other details
4. Mark as nominee if applicable
5. Upload supporting documents if required
6. Click "Save"

### Certifications
1. Go to Profile → Certifications tab
2. Click "Add Certification"
3. Enter certification name, issuing organization, issue date, expiry date
4. Upload certificate document
5. Add relevant skills
6. Click "Save"

### Academic Qualifications
1. Go to Profile → Education tab
2. Click "Add Qualification"
3. Enter degree, specialization, institution, passing year, percentage
4. Upload degree certificate
5. Click "Save"

## Leave Management

### Viewing Leave Balance
1. Navigate to "Leave Management"
2. View your current leave balance for all leave types
3. See accrued, consumed, available, and carried forward leaves

### Applying for Leave
1. Go to "Leave Management" → "Apply Leave"
2. Select leave type
3. Choose start date and end date
4. Enter number of days (auto-calculated)
5. Add reason/remarks
6. Upload supporting documents if required
7. Click "Submit"
8. Track approval status in "My Leave Requests"

### Editing Leave Request
1. Go to "My Leave Requests"
2. Find the pending request
3. Click "Edit" (only for pending requests)
4. Make changes
5. Click "Update"

### Withdrawing Leave Request
1. Go to "My Leave Requests"
2. Find the pending request
3. Click "Withdraw"
4. Confirm withdrawal

### Viewing Leave Calendar
1. Navigate to "Leave Calendar"
2. View your approved leaves on the calendar
3. See team members' leaves (if enabled)

## Attendance

### Viewing Attendance
1. Navigate to "Attendance"
2. View your attendance records
3. Filter by date range
4. See check-in/check-out times, working hours, status

### Regularization Requests
If you have attendance exceptions:
1. Go to "Attendance" → "Regularization"
2. Select the date
3. Choose exception type (late coming, short attendance, etc.)
4. Add reason and supporting documents
5. Submit for approval

## Work Schedule

### Viewing Your Roster
1. Navigate to "Work Schedule" → "Roster Calendar"
2. View your assigned shifts and work schedules
3. See shift timings and locations

### Roster Change Request
1. Go to "Work Schedule" → "Roster Change Requests"
2. Click "Request Change"
3. Select the date
4. Choose new shift (if applicable)
5. Add reason
6. Submit for approval

## Leave Encashment

### Checking Eligibility
1. Navigate to "Leave Encashment"
2. Select leave type
3. Enter number of days to encash
4. Click "Check Eligibility"
5. View eligible amount and rate per day

### Applying for Encashment
1. Go to "Leave Encashment" → "Apply"
2. Select leave type
3. Enter number of days
4. Add reason
5. Review calculated amount
6. Submit request
7. Track approval status

### Viewing Encashment History
1. Go to "Leave Encashment" → "My Requests"
2. View all your encashment requests
3. See status, approval details, and payroll processing status

## Reports

### Accessing Reports
1. Navigate to "Reports"
2. Select report type
3. Apply filters (date range, leave type, etc.)
4. Generate report
5. Export to Excel/PDF if needed

### Available Reports
- Leave Balance Report
- Leave Utilization Report
- Attendance Summary
- Attendance Exceptions
