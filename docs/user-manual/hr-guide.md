## Employee Management

### Employee Onboarding
1. Navigate to "Employees" → "Add Employee"
2. Fill in personal information
3. Set employment details (department, designation, joining date)
4. Configure work schedule
5. Set up leave balance
6. Create user account
7. Send welcome email

### Managing Employee Profiles
1. Go to "Employees" → Select employee
2. View and edit profile information
3. Manage family details, certifications, education
4. Approve profile update requests
5. Update official data (SAP-synced fields)

### Employee Offboarding
1. Navigate to "Employees" → "Offboarding"
2. Initiate offboarding process
3. Process final settlement
4. Calculate leave encashment
5. Update employee status

## Leave Management

### Leave Accrual Configuration
1. Go to "Leave Management" → "Accrual Policies"
2. Create or edit accrual policies
3. Set accrual frequency (monthly/yearly)
4. Configure pro-rata and carry forward rules
5. Apply to specific departments/designations

### Manual Leave Accrual
1. Navigate to "Leave Management" → "Accrual" → "Manual Trigger"
2. Select accrual type (monthly/yearly)
3. Choose month/year
4. Run accrual process
5. Review accrual results

### Leave Balance Management
1. View employee leave balances
2. Adjust balances manually if needed
3. Process year-end carry forward
4. Handle leave lapses

### Leave Approvals
1. Review pending leave requests
2. Approve or reject at HR level
3. Handle escalations
4. Process leave cancellations

### Leave Encashment
1. Configure encashment rules
2. Review encashment requests
3. Approve eligible requests
4. Process for payroll integration

## Best Practices

### Leave Management
- Configure accrual policies at the start of the year
- Review and approve leave requests promptly
- Monitor leave utilization patterns
- Process year-end carry forward accurately

### Attendance Management
- Regularly sync biometric data
- Process attendance exceptions promptly
- Maintain accurate attendance records
- Generate reports for payroll on time

### Compliance
- Monitor document expiry dates
- Track compliance deadlines
- Generate compliance reports regularly
- Ensure statutory compliance
