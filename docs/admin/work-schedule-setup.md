# Work Schedule Setup Guide

## Overview
This guide explains how to configure work schedules, shift templates, and rosters.

## Prerequisites
- Admin or HR role access
- Understanding of company work schedules

## Shift Templates

### Creating Shift Template
1. Navigate to "Work Schedule" → "Shift Templates"
2. Click "Create Template"

### Configuration
- **Template Name**: Enter name (e.g., "Day Shift", "Night Shift")
- **Start Time**: Shift start time
- **End Time**: Shift end time
- **Break Duration**: Break time in minutes
- **Working Hours**: Total working hours
- **Applicable Days**: Days of week
- **Location**: Shift location (if applicable)

### Save Template
Click "Save"

## Work Schedules

### Creating Work Schedule
1. Go to "Work Schedule" → "Schedules"
2. Click "Create Schedule"

### Assign Schedule to Employee
- **Employee**: Select employee
- **Shift Template**: Choose shift template
- **Effective From**: Start date
- **Effective To**: End date (optional)
- **Location**: Work location

### Save Schedule
Click "Save"

## Roster Management

### Creating Roster
1. Navigate to "Work Schedule" → "Roster Management"
2. Click "Create Roster"

### Roster Assignment
- **Date**: Select date
- **Employee**: Select employee
- **Shift**: Choose shift template
- **Location**: Work location

### Bulk Roster Upload
1. Prepare Excel file with format:
   - Employee Email
   - Date
   - Shift Template Name
   - Location
2. Go to "Roster Management" → "Bulk Upload"
3. Upload Excel file
4. Review and confirm

## Holiday Calendar

### Adding Holidays
1. Go to "Work Schedule" → "Holidays"
2. Click "Add Holiday"

### Holiday Configuration
- **Holiday Name**: Enter name
- **Date**: Holiday date
- **Location**: Location-specific (if applicable)
- **Type**: Public holiday, company holiday, etc.
- **Description**: Additional details

### Save Holiday
Click "Save"

## Roster Change Requests

### Approving Change Requests
1. Navigate to "Work Schedule" → "Roster Change Requests"
2. View pending requests
3. Approve or reject with comments

## Best Practices

1. **Standardize Shifts**: Create standard shift templates
2. **Plan in Advance**: Create rosters well in advance
3. **Use Bulk Upload**: Use Excel upload for efficiency
4. **Maintain Holiday Calendar**: Keep holiday calendar updated
5. **Handle Changes Promptly**: Process change requests quickly

## Troubleshooting

### Roster Not Displaying
- Check employee schedule assignment
- Verify shift template exists
- Review date range

### Bulk Upload Errors
- Verify Excel format
- Check employee emails
- Validate shift template names
